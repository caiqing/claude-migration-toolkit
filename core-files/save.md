# 保存协作会话

当你想要保存当前的AI协作会话时，使用此命令。它会自动调用协作会话管理系统的保存功能，生成结构化的Markdown文档并更新索引。

## 使用方法

/save

## 功能说明

此命令会：
- 检查是否有活跃的协作会话
- 保存完整的会话内容为Markdown文档
- 自动更新协作会话索引
- 清理临时状态文件

## 输出示例

**成功保存时：**
```
✅ 协作会话保存成功！

📁 文档位置: docs/collaboration/20251006-主题描述.md
📋 会话ID: session-20251006-1759747123
🔄 索引状态: 已自动更新
⏰ 保存时间: 2025-10-06 18:38:45
```

**无活跃会话时：**
```
⚠️ 当前没有活跃的协作会话

💡 使用建议：
1. 首先开始协作会话：/collaborate [范式] [主题]
2. 然后使用 /save 保存会话
```
